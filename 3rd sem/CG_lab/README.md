
# Computer Graphics Lab



## How to run

- Clone the repository

```bash
  git clone https://github.com/Yakumaa/CG_lab
```
- Compile the code
```bash
  g++ filename.cpp -o output -g -lGL -lGLU -lglut
```
- Run the executable
```bash
  ./output
```

## Or
- Run the build task
```bash
  ctrl+shift+b
```
- then run the executable file
```bash
  ./filename
```
